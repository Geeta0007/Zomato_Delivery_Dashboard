<h2>📖 Project Overview</h2>

<p>
The <b>Zomato Analytics Dashboard</b> is designed to analyze restaurant data across different countries and cities.
It enables users to explore restaurant performance, pricing patterns, customer ratings, cuisines, online delivery services,
and table booking availability through interactive visualizations and filters.
</p>
